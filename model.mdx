Aligned AI is a **compound product**: vision, structured routing, safeguards, and a main reasoning model work together per request. Benchmark pages plot **Aligned AI Fast** and **Aligned AI Research** as separate modes — not every underlying vendor model appears as its own point on the charts.

## What the benchmarks measure

Both modes are evaluated on the same **conversation-understanding** question set (comprehension, follow-ups, hallucination-sensitive prompts). Accuracy uses a 1–5 rubric aggregated across the suite.

Cost is normalized from public API list prices to an **average cost per chat** over the benchmark transcripts, so accuracy and cost share one axis on the [General benchmark](/general) scatter plot.
